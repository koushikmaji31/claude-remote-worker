# One fact per file

This `memory/` folder is long-term, durable memory for the remote worker. Put **one fact per
file** here. Every headless `claude -p` call loads `../CLAUDE.md`, and you reference these
files when relevant — so anything written here persists across calls and across the
laptop/cloud workers (the folder is synced via git).
